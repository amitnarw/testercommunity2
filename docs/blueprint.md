# **App Name**: Showcase Canvas

## Core Features:

- Interactive Demo Frame: Embed and display live demo URLs in an interactive iframe. The user can resize and interact directly with the demo within the app.
- Contextual Highlighting: Allow users to annotate and highlight specific elements within the demo frame to call out key features.
- Descriptive Note Window: A resizable window with descriptive text (generated by the application owner) pertaining to each frame of the showcase.
- AI-Powered Description Generation: Automatically generate a concise summary of the URL provided using a generative AI tool.

## Style Guidelines:

- Primary color: Vivid purple (#9D34FA) for a modern, tech-forward feel. The purple suggests creativity and innovation.
- Background color: Light gray (#EAE8ED) for a clean, professional canvas.
- Accent color: Electric blue (#43DDFF) to highlight interactive elements and call-to-action buttons.
- Body text: 'Inter', a sans-serif for clear, modern readability.
- Headline text: 'Space Grotesk', a sans-serif that pairs well with Inter to emphasize headings.
- Minimal, clear icons to represent key features and functions, avoiding excessive ornamentation.
- Clean, sectioned layout to clearly differentiate between the demo frame, description, and interactive tools.
- Subtle animations, such as smooth transitions when switching between demo frames, to enhance the user experience without distraction.
